## Цель курса

Целью курса является научиться проектировать и эффективно работать с реляционными базами данных: создавать таблицы, формировать связи, писать запросы, оптимизировать хранение и доступ к данным. В рамках курса практикуется написание разнообразных SQL-запросов (выборка, соединения, фильтрация, агрегаты и др.), освоение индексов и структур, обеспечивающих производительность и целостность данных. Кроме того, важной частью курса является взаимодействие с базой из программ на Python (через библиотеки и ORM).
